Import

import

import.js

Import using esm or TypeScript

Since

1.0.0

Example

import _d from 'datedash'
 
_d.date('3/14/2019', 'uk')
// => 14 Mar 2019
 
_d.addDays('3/6/19', 1, '-')
// => 03-07-2019
 
_d.subtractDays('3/6/19', 1, '-')
// => 03-05-2019
